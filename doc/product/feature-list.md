# 功能清单

## 功能地图

- 手机侧输入界面
- 文本发送到电脑
- 电脑端按键动作桥接
- 输入过程中的高频快捷操作
- 偏好设置与历史记录
- 面向 AI 编程场景的交互优化

## 核心功能

- 使用手机输入法进行文字输入与语音输入
- 将输入结果发送到电脑当前光标位置
- 在空输入时远程触发电脑上的 `Enter`
- 在空输入时远程触发电脑上的 `Backspace`
- 在需要时向电脑插入换行
- 让发送、回车、退格等高频操作尽量保持在输入法键盘附近完成

## 支撑功能

- 发送历史记录
- 主题与界面偏好设置
- 悬浮按钮显隐控制
- 草稿自动恢复
- Dock / 悬浮快捷操作栏

## 平台差异

- Web 版偏向“最轻量、最容易访问”的手机端入口
- Android 原生版用于进一步探索更稳定的输入、状态与系统能力接入
- 电脑端负责接收文本与按键动作，并将它们作用到当前光标所在位置
